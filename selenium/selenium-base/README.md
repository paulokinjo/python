# Selenium BASE

# Requirements
- python3
- virtualenv
- SeleniumBase
- ChromeDriver

# On the virtualenv
```
$ pip3 install -r requirements.txt
```

# Selenium Base Driver install
```
$ sbase install chromedriver latest

output:

Making [chromedriver 97.0.4692.71] executable ...

```
